# Code Lint 手順

コードの品質を維持し、バグの混入を防ぐために、以下の手順を順に実行してください。

## 1. フォーマットとリント (Ruff)

まず、コードのフォーマットを整え、基本的な構文エラーやスタイル違反を修正します。

```bash
# コードのフォーマット
uv run ruff format .

# リントチェックと自動修正
uv run ruff check . --fix
```

## 2. 型チェック (Pyright)

Pythonの型ヒントに基づき、厳密な型チェックを行います。

```bash
uv run pyright
```

## 3. テスト実行 (Pytest)

変更が既存の機能を壊していないか、単体テストを実行して確認します。

```bash
uv run pytest
```

## エラーが発生した場合

上記のいずれかのステップでエラーが発生した場合は、**必ずエラーを解消してから**次のステップやコミットに進んでください。
